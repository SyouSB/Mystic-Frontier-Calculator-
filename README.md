# Mystic Frontier Calculator (with OCR)

> "You don't need to enter anything!"

**[Launch Calculator](https://syousb.github.io/Mystic-Frontier-Calculator-/)**

## ✨ Features

*   **Fully Automated**: No manual input required. Just share your screen.
*   **Privacy First**: User data is **never stored**. Everything runs entirely locally in your browser.
*   **Real-time Analysis**: Instantly calculates Dice Total and Final Multiplier.

## 🚀 How to Use

1.  Open the **Mystic Frontier** UI in MapleStory.
2.  Go to the [Calculator Website](https://syousb.github.io/Mystic-Frontier-Calculator-/).
3.  Click the **"Select Screen"** button.
4.  Choose your **MapleStory window** or screen.
5.  That's it! The calculator will overlay the results automatically.

## 🖥️ Supported Resolutions

For the best detection accuracy, please use the recommended resolution:

*   ✅ **1366 x 768 (Recommended)**
*   ✅ **1920 x 1080**
*   ✅ **2560 x 1440**

## ⚠️ Important Notes & Limitations

*   **Window Position**: Do **not** move the Mystic Frontier UI window inside the game. It must be in its default position.
    *   *Fix:* If you moved it, close and reopen the Familiar window to reset its position.
*   **Occlusion**: Ensure your mouse cursor or other windows do not cover the Dice or Attribute text areas.
*   **Work in Progress**: This is a Beta version.
    *   Performance optimizations are ongoing to ensure smooth operation on various systems.
